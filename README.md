# NxtBuild 🚀

AI-powered web app builder that converts natural language into fully working web applications.

## Tech Stack
- React + Vite
- Node.js + Express
- MongoDB
- Google Gemini AI

## Features
- AI code generation
- Live preview
- Chat-based builder
- Project management
- Authentication (JWT)

## Run Locally

### Server
cd server
npm install
npm run dev

### Client
cd client
npm install
npm run dev
